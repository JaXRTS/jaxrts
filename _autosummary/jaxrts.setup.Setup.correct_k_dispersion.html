jaxrts.setup.Setup.correct_k_dispersion

Setup.correct_k_dispersion: bool

In an experiment, the value of k is dependent on the measured energy at this position on the detector. When including this effect, the returned spectrum will, however, violate detailed balance. It might therefore be instructive to set this value (which is True be default) to False to #: check, e.g. temperature differences when applying ITCF analysis methods.
