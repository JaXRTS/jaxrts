jaxrts.units.to_array

jaxrts.units.to_array(obj) → Array[source]

Create an array of an object. This gets around the annoying fact that a jax array cannot easily be created from a list of Quantities. Rather, one has to strip every entry of it’s unit and multiply it, after.
