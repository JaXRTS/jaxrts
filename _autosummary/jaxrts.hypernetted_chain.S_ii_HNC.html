jaxrts.hypernetted_chain.S_ii_HNC

jaxrts.hypernetted_chain.S_ii_HNC(k: Quantity, pdf, ni, r)[source]

Calculates the static structure factor for an isotropic system from the pair distribution function obtained used the HNC approach.

Note

Due to the numerical implementation of sinft(), we force the entry S_ii[0] to the value S_ii[1], as it would be always unity, otherwise.
